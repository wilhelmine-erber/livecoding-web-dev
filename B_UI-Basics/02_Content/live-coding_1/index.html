Ordered and unordered lists

Ordered lists are used when things need to happen in a specific order

  1. wake up
  2. make breakfast
  3. get dressed
  4. start coding

Unordered lists are ideal when order is not important, for example a grocery list

  • mineral water
  • pears
  • toothpaste
  • gummi bears
  • pretzels
