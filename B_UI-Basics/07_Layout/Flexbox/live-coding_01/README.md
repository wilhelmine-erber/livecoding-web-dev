## UIB - Layout - Flexbox - Live Coding 01

---

### Overview

Flexbox introduction focusing on the various properties available to use on the parent element. The learner should have a firm understanding of the main principles of Flexbox, primarily the importance of the main and cross axis and the difference between the flex parent (container) and children (flex items).

### Main Concepts

- Flex is best suited for components and small-scale layouts
- Flex parent - child relationship
- Wrapping and direction
- Aligning and justifying
